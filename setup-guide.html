Todo gratis. Seguí estos pasos en orden — sin terminal.

Buenas noticias: el código, la base de datos (Supabase) y las funciones de IA ya están creados y cargados. Solo faltan unas claves y permisos que se configuran haciendo clic en paneles. La app funciona incluso sin todo esto (con respaldos), pero completalo para tener IA, login con Google y notificaciones.

1 Subir el código a GitHub

Para que Vercel lo pueda importar y publicar.

  1. Creá un repositorio nuevo en GitHub (privado está bien).
  2. Subí esta carpeta (o conectá la rama actual).

2 Publicar en Vercel hosting gratis

Vercel hostea la web en un dominio gratuito *.vercel.app.

  1. Entrá a vercel.com/new e importá el repo de GitHub.
  2. En Environment Variables, pegá las del archivo .env.example (ver pasos 3–6 para los valores).
  3. Deploy. Anotá tu URL final, ej: https://brote.vercel.app.

3 URLs de redirección importante

Esto es lo que hace que los correos (confirmación / enlace mágico) apunten a tu web real y no a localhost.

  1. En Supabase → Authentication → URL Configuration:
    Site URL: https://TU-APP.vercel.app
    Redirect URLs (agregá ambas):
    https://TU-APP.vercel.app/**
    http://localhost:3000/** (para desarrollo local)
  2. En Vercel → Environment Variables, poné NEXT_PUBLIC_APP_URL = https://TU-APP.vercel.app. La app usa esto para armar los links de los correos.
  3. Login con email + contraseña y enlace mágico ya funcionan. Si querés que el alta sea instantánea sin confirmar correo, en Authentication → Providers → Email desactivá "Confirm email".

4 Login con Google opcional

Agrega el botón "Continuar con Google" (el login por correo ya anda sin esto).

  1. En Supabase → Authentication → Providers → Google, activalo.
  2. Creá credenciales OAuth en Google Cloud Console y pegá el Client ID/Secret. Usá como redirect el que te muestra Supabase.

5 Clave de IA (Google Gemini) gratis

Verifica fotos, recomienda acciones y resume noticias. Sin esto, la app usa respaldos automáticos.

  1. Conseguí una clave gratis en Google AI Studio.
  2. En Supabase → Edge Functions → Secrets, agregá:
    GEMINI_API_KEY = tu clave

6 Notificaciones push (VAPID) opcional

Permiten avisar "tu racha está por terminar", etc.

  1. Generá las claves: en la carpeta del proyecto corré npm run vapid (te da una pública y una privada).
  2. Agregá en Vercel → Environment Variables:
    NEXT_PUBLIC_VAPID_PUBLIC_KEY = clave pública
  3. Agregá en Supabase → Edge Functions → Secrets:
    NEXT_PUBLIC_VAPID_PUBLIC_KEY = clave pública
    VAPID_PRIVATE_KEY = clave privada
    VAPID_SUBJECT = mailto:tu@correo.com

7 Clave de servicio para tareas automáticas opcional

Para que se actualicen las noticias y se envíen las push automáticamente (cron).

  1. En Supabase → Project Settings → API copiá la service_role key (¡es secreta!).
  2. Pegala en Vercel → Environment Variables:
    SUPABASE_SERVICE_ROLE_KEY = service_role key
  3. Guardala también en Supabase → Vault (Database → Vault → New secret):
    Nombre: service_role_key · Valor: la misma key
    Esto habilita el cron de noticias y el disparo de notificaciones push.
